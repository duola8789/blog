# blog
个人博客源码

## 命令

基本命令：
- `hexo init` - 初始化博客
- `hexo n [new blog]` - 新建文章
- `hexo g` - 生成网页
- `hexo s` - 启动服务器预览
- `hexo d` - 部署
- `hexo clean ` - 清除缓存

常用命令：

- `npm run dev` - 服务器预览（localhost:4000）
- `hexo n [blog_name]` - 新建文章
- `hexo n [page/draft/post] [page_name] ` - 新建页面/草稿/文章
- `npm run deploy` - 清除缓存并
